# 01. 목표와 범위

## 1.1 목표

- **게임**: 길드원이 던전을 자동 탐험하는 길드 관리형 게임
- **역할**: 사용자는 **길드 마스터**. 아지트에서 길드원과 대화하며 상황을 파악하고 지시한다
- **대화**: 길드원마다 **성격·배경·최근 던전 경험**이 반영된, 자연스러운 대화. **데이터를 기반으로** (RAG처럼 검색·조합한 컨텍스트만 보고) 말하게 설계함

## 1.2 범위 (이번 미니프로젝트)

| 포함 | 비포함 |
|------|--------|
| 길드 마스터 ↔ 길드원 1:1 대화 | 실제 던전 플레이/자동 탐험 |
| 길드원 ↔ 길드원 대화 (동료 대화) | Unity 연동 |
| 성격 수치에 따른 말투·반응 차이 | 전투/아이템 상세 로직 |
| 캐릭터 배경·던전 로그를 맥락으로 활용 | 멀티플레이어/네트워크 |
| 로컬 LLM(Ollama)으로 대사 생성 | 상용 API(OpenAI 등) |

## 1.3 성공 기준

- 대화가 **사람끼리 말하는 느낌**으로 이어짐
- **주제가 유지**되고 (유물/포션/함정 등) 엉뚱한 말로 새지 않음
- 캐릭터마다 **말투·가치관**이 구분됨
- **배경·던전 로그**를 기억하고 언급할 수 있음
